# strategic-employee-attrition-analysis
Data-driven approach to understanding employee turnover and building predictive models to support strategic HR decision-making.

📌 Contexto del problema
La rotación de empleados representa uno de los principales costos ocultos para las organizaciones. La pérdida de talento implica gastos de reclutamiento, entrenamiento y pérdida de conocimiento organizacional.

En este proyecto se analiza un dataset de recursos humanos con 1470 empleados, con el objetivo de identificar qué factores organizacionales, laborales y de trayectoria profesional se asocian con la decisión de abandonar la empresa.

🎯 Objetivo del análisis
La rotación de empleados representa uno de los principales costos ocultos para las organizaciones. La pérdida de talento implica gastos de reclutamiento, entrenamiento y pérdida de conocimiento organizacional.

En este proyecto se analiza un dataset de recursos humanos con 1470 empleados, con el objetivo de identificar qué factores organizacionales, laborales y de trayectoria profesional se asocian con la decisión de abandonar la empresa.

📊 Principales hallazgos
El análisis exploratorio sugiere que varios factores se asocian con la rotación laboral:
Mayor estancamiento profesional (Stagnation Index) se relaciona con mayor probabilidad de attrition.
Menor velocidad de promoción (Promotion Velocity) se asocia con mayor abandono.
Menor tasa de lealtad organizacional (Loyalty Rate) indica trayectorias laborales más móviles.
Variables clásicas de People Analytics como Overtime, DistanceFromHome y MonthlyIncome también muestran asociación con attrition.

🤖 Performance del modelo
Se entrenó un modelo de regresión logística para estimar la probabilidad de abandono.
El modelo permite:
identificar variables relevantes en la predicción
estimar probabilidades individuales de attrition
explorar patrones de riesgo de rotación en la organización

📈 Implicancias estratégicas
Los resultados sugieren que la rotación laboral no depende solo de factores demográficos, sino también de trayectorias profesionales dentro de la organización.

Indicadores como:
velocidad de promoción
movilidad de carrera
estancamiento profesional

pueden ser útiles para diseñar estrategias de retención de talento y monitorear el desarrollo profesional de los empleados.
